Para poder ver el y correr localmente la aplicación, favor de usar el GITHUB: 
https://github.com/PabloCoRz/duckybiblioteca (WEB)

Pasos para poder correrlo:
1. Haz un git clone en tu computadora.
2. Corre "npm install" (solamente la primera vez) en la terminal.
3. Corre "npm run dev" en la terminal.
4. Entra a http://localhost:3000 (verifica que sea el correcto)
